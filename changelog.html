Changelog

Version history and release notes for AirShare

v0.1.3 Latest

October 12, 2025

Bug Fixes

  • • Critical: Fixed peer discovery bug where Mac and Windows couldn't see each other due to hostname collision. Now uses unique UUID (app_id) for reliable peer identification.
  • • UI: Fixed toast notifications showing 3 times when changing discoverability settings. Added debounce mechanism.
  • • Updater: Fixed hardcoded version number in update dialog. Now shows actual current version dynamically.

Improvements

  • • Performance: Removed 72 excessive console/terminal log statements. Cleaner logs, better performance.
  • • Network: Improved peer discovery reliability using UUID-based identification instead of hostname comparison.
  • • Code Quality: Fixed Rust formatting issues (35 trailing whitespace errors resolved).

Documentation

  • • All documentation translated to English
  • • Removed redundant documentation files (consolidated into core docs)
  • • Added comprehensive GitHub Actions setup guide
  • • Created FAQ, Licenses, and Changelog pages

DevOps

  • • Setup GitHub Actions for automated multi-platform releases
  • • Configured CI workflow for code quality checks (PR-only)
  • • Removed Dependabot (manual version control preferred)
  • • Added helper scripts for version bumping and secret setup

Download: Available for macOS (Apple Silicon & Intel), Windows, and Linux from GitHub Releases

More updates coming soon! Follow us on GitHub for the latest news.
