1 Getting Started

How do I install AirShare?

AirShare is available for Windows, macOS, and Linux. Download the installer from our website and follow the installation wizard. No additional setup is required - just install and run!

  • Windows: Download the .exe installer
  • macOS: Download the .dmg file and drag to Applications
  • Linux: Download the .AppImage file and make it executable

How do I share files with AirShare?

Sharing files is simple:

  1. Make sure AirShare is running on both devices
  2. Both devices must be on the same WiFi network
  3. Drag and drop files onto a device in the radar, or click on a device and select files
  4. Enter the PIN code when prompted
  5. Files will be transferred securely

Why can't I see other devices on the radar?

If you can't see other devices, check these common issues:

  • Ensure both devices are on the same WiFi network
  • Check that AirShare is running on both devices
  • Verify that "Discovery" is set to "Everyone" in preferences
  • Check your firewall settings - AirShare needs network access
  • Try refreshing the device list (F5 or View → Refresh)

2 Troubleshooting

Transfer failed or stuck - what should I do?

If a transfer fails or gets stuck:

  • Check that both devices are still connected to the same WiFi
  • Ensure both devices have AirShare running
  • Try canceling and restarting the transfer
  • Check available disk space on the receiving device
  • Restart AirShare on both devices if the problem persists

AirShare won't start or crashes

If AirShare won't start or crashes frequently:

  • Check that you have the latest version installed
  • Restart your computer
  • Check if antivirus software is blocking AirShare
  • Try running as administrator (Windows) or with sudo (Linux)
  • Check the log files in the AirShare data directory

Slow transfer speeds

Transfer speeds depend on several factors:

  • WiFi network quality and signal strength
  • Network congestion and other devices using bandwidth
  • Device performance (CPU, RAM, storage speed)
  • File size and type (large files may take longer to process)
  • Distance between devices

For best performance, ensure devices are close to the WiFi router and minimize other network activity.

3 Features & Pro Version

What's the difference between Free and Pro versions?

Free Version

  • • Up to 200MB file size limit
  • • 10 transfers per day
  • • 7 days transfer history
  • • Basic file sharing

Pro Version

  • • Unlimited file sizes
  • • Unlimited daily transfers
  • • 365 days transfer history
  • • Mobile Share (QR code)
  • • Priority support

How do I upgrade to Pro?

To upgrade to Pro:

  1. Go to Device → Activate License in the menu
  2. Enter your Pro license key
  3. Click "Activate Pro"
  4. Restart AirShare to apply changes

You can purchase a Pro license from our website or contact support for enterprise licensing.

What is Mobile Share?

Mobile Share is a Pro feature that allows you to share files with mobile devices using QR codes:

  • Generate a QR code on your computer
  • Scan the QR code with your phone's camera
  • Access a web interface to send/receive files
  • Works with any device that has a web browser

4 Security & Privacy

Is AirShare secure?

Yes, AirShare uses multiple security measures:

  • QUIC Protocol: All transfers use encrypted QUIC connections
  • PIN Authentication: Each transfer requires a PIN code
  • Peer-to-Peer: Files never pass through external servers
  • Local Network Only: Transfers only work on your local WiFi
  • No Data Collection: We don't collect or store your files

Can others on my network see my files?

No, your files are protected by:

  • End-to-end encryption during transfer
  • PIN code authentication required for each transfer
  • Files are only accessible to devices you explicitly choose
  • No files are stored on network infrastructure

Only devices with AirShare installed and the correct PIN can access your files.

What happens to my files after transfer?

Files are handled securely:

  • Files are saved to the recipient's chosen download folder
  • No copies are kept on the sender's device after transfer
  • Transfer history is stored locally (not uploaded anywhere)
  • You can clear transfer history at any time

5 Technical Details

What operating systems are supported?

AirShare supports:

  • Windows: Windows 10 and later (64-bit)
  • macOS: macOS 10.14 (Mojave) and later
  • Linux: Most modern distributions (Ubuntu, Fedora, etc.)

Mobile versions are planned for future releases.

What network requirements does AirShare have?

AirShare requires:

  • WiFi network connection (wired Ethernet also works)
  • All devices must be on the same local network
  • No internet connection required for transfers
  • Ports 8080-8090 should be available (configurable)
  • mDNS/Bonjour support for device discovery

How do I check the log files?

Log files are located in:

  • Windows: %APPDATA%\AirShare\logs\
  • macOS: ~/Library/Application Support/AirShare/logs/
  • Linux: ~/.local/share/AirShare/logs/

Log files contain technical details about transfers and can help diagnose issues.

6 Support & Contact

How can I get help?

We offer several support options:

  • Email Support: Contact us through the app (Help → Contact)
  • Bug Reports: Use the bug report feature in the app

How do I report a bug?

To report a bug effectively:

  1. Go to Help → Contact in the app
  2. Select "Bug Report"
  3. Describe what happened and when
  4. Include your operating system and AirShare version
  5. Attach log files if possible

Can I request new features?

Yes! We welcome feature requests:

  • Describe the feature and how it would help you
  • Check our GitHub repository for existing feature requests
