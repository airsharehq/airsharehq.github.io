Frequently Asked Questions

Everything you need to know about AirShare

General

What is AirShare?

AirShare is a cross-platform file sharing app that lets you transfer files instantly over your local network. Think AirDrop, but for macOS, Windows, and Linux. Uses modern QUIC protocol with TLS 1.3 encryption for fast, secure transfers.

Is it free?

Yes! AirShare is free for personal use with limits (200MB file size, 10 transfers/day). Pro version removes all limits and adds unlimited transfer history. One-time purchase, no subscriptions.

Do I need an account?

No! AirShare works completely offline. No accounts, no cloud, no telemetry. Just install and start sharing.

Is my data secure?

Yes! All transfers use TLS 1.3 encryption. Files never leave your local network and are transferred directly peer-to-peer. SHA-256 hash verification ensures file integrity.

Setup & Usage

How do I install AirShare?

macOS: Download the .dmg file, open it, and drag AirShare to Applications folder.

Windows: Download the .msi installer and run it. Allow through firewall when prompted.

Linux: Download .AppImage, make it executable (chmod +x), and run.

Why can't I see other devices?

Common causes:

  • Different networks - Both devices must be on same Wi-Fi
  • Firewall blocking - Allow AirShare through firewall (especially Windows)
  • Network type (Windows) - Network must be set to "Private" not "Public"
  • Discoverability - Check Settings → set to "Everyone"

Quick fix: Restart AirShare on both devices and toggle Discoverability to Nobody → Everyone.

How do I send a file?

Simply drag and drop files onto any device that appears in your Radar. Or click the device and select files. Transfers start immediately with real-time progress tracking.

Can I send multiple files at once?

Yes! Select multiple files or drag entire folders. AirShare will queue them and transfer efficiently.

Technical

What ports does AirShare use?

AirShare uses:

  • mDNS: Port 5353 (UDP) - for device discovery
  • QUIC: Ports 59875-59925 (UDP) - for file transfers

Port selection is deterministic based on device UUID, ensuring stable peer identity.

What is QUIC protocol?

QUIC is a modern transport protocol developed by Google, now standardized by IETF. It's faster than TCP, has built-in encryption (TLS 1.3), and handles packet loss better. Perfect for local file transfers!

Is AirShare open source?

The core app is proprietary, but we use many open-source libraries (see Licenses). We're committed to transparency and security.

Does it work on corporate networks?

It depends. AirShare works on most home and small office networks. Corporate networks with strict firewall rules or VLAN isolation may block mDNS multicast. Contact your IT admin if discovery doesn't work.

Pro:

  • Unlimited file size
  • Unlimited transfers
  • Unlimited history

Can I transfer folders?

Currently, you can select multiple files from a folder. Full folder transfer support is coming in a future update.

Does it support resuming interrupted transfers?

Not yet. If a transfer is interrupted, you'll need to restart it. Resume support is planned for a future release.

Can I use AirShare over the internet?

No. AirShare is designed specifically for local network transfers. It uses mDNS which only works on the same local network. This is a security feature - your files never leave your local network.

Troubleshooting

Transfer failed or stuck at 0%

Try these fixes:

  • Cancel and retry the transfer
  • Check that both devices are still on the same network
  • Restart AirShare on both devices
  • Check firewall isn't blocking ports 59875-59925
Windows can't see Mac (or vice versa)

On Windows:

  • Windows Security → Firewall → Allow AirShare
  • Network settings → Set to "Private" network

On Mac:

  • System Settings → Privacy & Security → Local Network → Enable AirShare
  • If using firewall, allow AirShare through it
App won't start or crashes immediately

Try these steps:

  1. Completely quit AirShare (check system tray)
  2. Delete app data folder and restart
  3. Reinstall AirShare
  4. Check system requirements (macOS 11+, Windows 10+)
Where are my received files?

By default, files are saved to your Downloads folder. You can change this in Settings → General → Download Folder.

Privacy & Security

Can someone intercept my files?

Extremely unlikely. All transfers use TLS 1.3 encryption (same as HTTPS). Even on the same local network, transfers are encrypted end-to-end. Only the receiving device can decrypt the files.

Does AirShare collect any data?

No. AirShare has zero telemetry, zero analytics, zero tracking. Everything stays on your device. We never see your files, transfer history, or usage patterns.

Where is my data stored?

All data is stored locally on your device:

  • macOS: ~/Library/Application Support/io.github.jaksatomovic.airshare
  • Windows: %APPDATA%\io.github.jaksatomovic.airshare
  • Linux: ~/.local/share/io.github.jaksatomovic.airshare

Licensing

How do I activate Pro license?

After purchase, you'll receive a license key via email. Open AirShare → Settings → License → Enter your key. Activation is instant and works offline after first verification.

Can I use my license on multiple devices?

No. Each license is valid for one device only. If you need AirShare Pro on multiple devices, you'll need to purchase a separate license for each device. If you have issues or need to transfer a license to a new device, please contact support.

What's your refund policy?

All sales are final. We do not offer refunds for one-time or lifetime licenses, except where required by law. Please try the free version first to ensure AirShare meets your needs before purchasing Pro.
